AI Personal Finance Tracker

An AI-powered personal finance management application built with Python, Streamlit, SQLite, Hugging Face Transformers, and Google Gemini.

Features
Upload transaction CSV files
Automatic transaction categorization using NLP
Expense analytics and visualization
Budget tracking
Monthly spending trends
AI-generated financial insights
Download reports
Tech Stack
Python
Streamlit
SQLite
Hugging Face Transformers
Google Gemini API
Pandas
Matplotlib
Installation
pip install -r requirements.txt
streamlit run app.py
